# Mojo

## Overview

Mojo是运行时库的集合，用于提供一种平台无关的IPC原语的抽象、消息IDL的格式以带有多种目标语言的代码生成器的Bindings库。有了以上的组件，那么可以实现任意进程内部或者跨进程的消息传递服务。
